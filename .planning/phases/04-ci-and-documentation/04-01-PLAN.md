# Plan 01: CI Workflow Modernization

## Objective

Modernize `.github/workflows/main.yml` to validate the full PHP 8.3+/Laravel 11+ matrix with correct testbench pinning and Pest as test runner. Fix the broken cache step and upgrade action versions to v4.

## Context

Current workflow state:
- PHP matrix: [8.1, 8.2, 8.3] with excludes for Laravel 11/12 + PHP 8.1
- Laravel matrix: [10.*, 11.*, 12.*]
- Actions: checkout@v3, cache@v3
- Cache step references `steps.composer-cache.outputs.dir` but no step has `id: composer-cache`
- Test runner: `vendor/bin/phpunit`
- No testbench pinning — relies on composer constraint resolution

    1. Update workflow name from "Unit Test" to "Tests"

    3. Rename job from "build" to "tests"
    4. Replace PHP matrix: `[8.3, 8.4]` (remove 8.1, 8.2)
    5. Replace Laravel matrix: `['11.*', '12.*']` (remove 10.*)
    6. Keep dependency-version: `[prefer-lowest, prefer-stable]`
    7. Remove the `exclude:` block entirely (all combinations valid)
    8. Add `include:` block to pin testbench per Laravel version:
       ```yaml
       include:
         - laravel: '11.*'
           testbench: '^9.0'
         - laravel: '12.*'
           testbench: '^10.0'
       ```
    9. Update job name format: `PHP ${{ matrix.php }} - L${{ matrix.laravel }} - ${{ matrix.dependency-version }}`
    10. Update `actions/checkout@v3` to `actions/checkout@v4`
    11. Add `coverage: none` to setup-php (no coverage needed)
    12. Add new step with `id: composer-cache` that runs `echo "dir=$(composer config cache-files-dir)" >> $GITHUB_OUTPUT`
    13. Update `actions/cache@v3` to `actions/cache@v4` with correct path reference
    14. Update Install step to also require testbench: `composer require "laravel/framework:${{ matrix.laravel }}" "orchestra/testbench:${{ matrix.testbench }}" --no-interaction --no-update`
    15. Rename "Run PHPUnit tests" to "Run tests" and change `vendor/bin/phpunit` to `vendor/bin/pest`

    Complete file content:
    ```yaml
    name: Tests

    on:
      pull_request:
        branches:
          - master
          - main
      push:
        branches:
          - master
          - main

    jobs:
      tests:
        runs-on: ubuntu-latest

        strategy:
          fail-fast: false
          matrix:
            php: [8.3, 8.4]
            laravel: ['11.*', '12.*']
            dependency-version: [prefer-lowest, prefer-stable]
            include:
              - laravel: '11.*'
                testbench: '^9.0'
              - laravel: '12.*'
                testbench: '^10.0'

        name: PHP ${{ matrix.php }} - L${{ matrix.laravel }} - ${{ matrix.dependency-version }}

        steps:
          - name: Checkout
            uses: actions/checkout@v4

          - name: Setup PHP
            uses: shivammathur/setup-php@v2
            with:
              php-version: ${{ matrix.php }}
              coverage: none

          - name: Get Composer cache directory
            id: composer-cache
            run: echo "dir=$(composer config cache-files-dir)" >> $GITHUB_OUTPUT

          - name: Cache Composer dependencies
            uses: actions/cache@v4
            with:
              path: ${{ steps.composer-cache.outputs.dir }}
              key: ${{ runner.os }}-composer-${{ hashFiles('**/composer.json') }}
              restore-keys: ${{ runner.os }}-composer-

          - name: Install Composer dependencies
            run: |
              composer require "laravel/framework:${{ matrix.laravel }}" "orchestra/testbench:${{ matrix.testbench }}" --no-interaction --no-update
              composer update --${{ matrix.dependency-version }} --prefer-dist --no-interaction

          - name: Run tests
            run: vendor/bin/pest
    ```
  </action>
