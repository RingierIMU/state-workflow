<task type="auto">
  <name>Task 1: Install Rector and create rector.php configuration</name>
  <files>composer.json, composer.lock, rector.php</files>
  <action>
1. Install Rector and the Laravel extension:
   ```bash
   composer require --dev rector/rector driftingly/rector-laravel
   ```

2. Create `rector.php` in the project root with this configuration:
   ```php
   <?php

   declare(strict_types=1);

   use Rector\Config\RectorConfig;
   use RectorLaravel\Set\LaravelSetList;

   return RectorConfig::configure()
       ->withPaths([
           __DIR__ . '/src',
       ])
       ->withPhpSets(php83: true)
       ->withSets([
           LaravelSetList::LARAVEL_110,
       ])
       ->withPreparedSets(
           deadCode: true,
           typeDeclarations: true,
       )
       ->withSkip([
           // Protect interface method signatures from type additions — these are implemented externally
           __DIR__ . '/src/Interfaces',
       ]);
   ```

   **Key configuration choices:**
   - `withPaths([__DIR__ . '/src'])` — RECT-01: only src/, never tests/
   - `withPhpSets(php83: true)` — enables all PHP 8.0 through 8.3 modernization rules
   - `LaravelSetList::LARAVEL_110` — Laravel-specific modernizations
   - `withPreparedSets(deadCode: true, typeDeclarations: true)` — dead code removal and type declarations
   - `withSkip([__DIR__ . '/src/Interfaces'])` — protect externally-implemented interface signatures

3. Verify Rector is installed and config is valid:
   ```bash
   vendor/bin/rector --version
   vendor/bin/rector process --dry-run --no-progress-bar 2>&1 | head -5
   ```
  </action>
